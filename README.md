# it-challenge-frontend

Tu budem dávať inštrukcie na run-ovanie backendu a podobné veci. Očakávajte gramatické chybya

# Setup

Na youtube si nájdite ako sa inštaluje Node.js a Vite, a toto spravte najprv.

Príkazy runnujte v itChallengeFrontend (čiže v terminali "cd itChallengeFrontend". ak niekto npm installne 5 tisíc modulov do root directory tak mi musí dať 10€)

# Spojenie GIT a VSCode:

(Očakávam že si logged in do GitHub účtu na VSCode. Ak nie, najdi youtube tutoriál, skill issue) 

1. Klikni Source Control (tretí tab od hora, v sidebar-e na ľavo)
2. Click "Initialise Repository"
3. Objaví sa "Changes". Hoverni na to, objavia sa 3 bodky napravo od toho. Remote -> Add Remote
4. Promptne ťa možno o link, možno o prijamy výber. Tam daj URL na toto repo (https://github.com/shiftyboi1/it-challenge-frontend). Ak toto nefunguje, jeden z nás je debil.
5. Poprosí vas o name, MUSÍTE ho pomenovať "origin"
6. Si connected. Teraz, pod "Changes" nájdeš graph. **Najprv klikni FETCH, potom PULL. TOTO JE POVINNÉ. Ak toto nespravíš, dlžíš mi 10€**

# Important 

Neupravujte rovnaký súbor naraz. ďakujem.
Po inštalácii musíte stiahnuť moduly. To spravíte príkazom "npm install".
Lokálny server spustíte príkazom "npm run dev"
